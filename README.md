# Hotel Harsha, Shimla - Official Website

![Hotel Harsha Banner](I91A9638.jpg)

A premium, high-conversion landing page developed for **Hotel Harsha, Shimla**, a heritage establishment founded in 1977. This project blends Victorian architectural heritage with modern web standards to create a seamless luxury experience.

## 🌟 Features
- **Modern Minimalist UI**: Clean, high-contrast design using Tailwind CSS and Bootstrap 5.
- **Smart Media Experience**: Background hero video with "Smart Unmute" logic and interactive room carousels.
- **Dual-View Gallery**: Instant toggle between immersive slideshows and masonry collages.
- **Direct Reservations**: Integrated WhatsApp booking system and Web3Forms contact hub.
- **Fully Responsive**: Optimized for high performance on both mobile and desktop.
